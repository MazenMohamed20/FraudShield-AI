# 📌 Overview

FraudShield AI is a machine learning-based fraud detection platform designed to identify suspicious credit card transactions.

The project follows a complete ML workflow:

- Data preprocessing
- Handling imbalanced data
- Model training
- Model evaluation
- Deployment through an interactive dashboard


The system allows users to upload transaction datasets, analyze transactions, detect fraud, view probability scores, and download prediction reports.

# ⚙️ Installation


Clone the repository:

```bash
git clone https://github.com/USERNAME/FraudShield-AI.git
```


Navigate to project:

```bash
cd FraudShield-AI
```


Install dependencies:

```bash
pip install -r requirements.txt
```


Run application:

```bash
streamlit run app/app.py
```
